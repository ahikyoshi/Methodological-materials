Маркированные списки

Описание

Нужны для обозначения нумерованного списка

Пример маркированных списков:

Базовое значение

  • Банан
  • Тыква
  • Арбуз
  • Снова тыква

Круговой список: circle

  • Банан
  • Тыква
  • Арбуз
  • Снова тыква

Квадратный список: square

  • Банан
  • Тыква
  • Арбуз
  • Снова тыква

Без метки: none

  • Банан
  • Тыква
  • Арбуз
  • Снова тыква

Нумерованные списки

Пример нумерованных списков:

Базовые значения

  1. Банан
  2. Тыква
  3. Арбуз
  4. Снова тыква

Римские цыфры: I

  1. Банан
  2. Тыква
  3. Арбуз
  4. Снова тыква

Буквенный список: a

  1. Банан
  2. Тыква
  3. Арбуз
  4. Снова тыква
