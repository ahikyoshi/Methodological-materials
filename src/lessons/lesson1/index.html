Настройки Visual studio code

Мы используем программу Visual studio code не просто так, это наш самый верный помощник при написании страниц. Но, чтобы она начал нам помогать, необходимо помочь ему понять что мы хотим. Для этого мы должны настроить следующие:

  • Auto Save - снимает с нас задачу постоянно сохранять документ, и сохраняет его сам при смене окна. Нужно поставить режим: OnWindowChange
  • Format on save - делает наш документ красивым при каждом сохранении. Нужно поставить галочку.

Расширения

Это маленькие помощники, которые и будут устанавливаться и помогать нам работать. Я приведу примеры, тех расширений которые будут установленны у вас в программе в академии

  • Live server - позволяет вам не обновлять вручную страницу, после каждого сохранения файла. Что сильно ускоряет работу над проектом. Для запуска найдите ваш index.html > ПКМ > Open with Live server
  • Auto Rename Tag - когда вы пытаетесь изменить свой тег, его нужно менять и при открытии и при закрытии. Это расширение, старается сразу изменить содержимое и открывающего и закрывающего тега
  • CSS Peek - позже когда вы будете изменять стили страницы, вам понадобиться искать блоки по именам, это расширение в будущем очень поможет нам сразу переходить на нужные блоки
  • HTMLHint - по умолчанию ошибки которые вы допускаете в коде не показывает, и в конечном итоге мы можем запутаться в них. Для этого нам и нужно расширение которое поможет нам отслеживать их. Расширение рисует желтую линию под ошибкой.
  • Material Icon Theme - Делает более приятные иконки. Нам не сильно нужно, просто пример того что программу можно изменять под себя.
  • Prettier - Code formatter - самое полезное расширение. С самого начала дисциплины я буду говорит, что очень важно следить за тем, чтобы все правильно и чисто выглядело. Но как за час запомнить все этим правила? Сделаем как профи, а именно поставим помощника который сделает это за нас. Prettier - занимается тем, что делает ваш код красивым и чистым без вашего участия.

HTML5

Что это?

HTML расшифровывается как – HyperText Markup Language (язык гипертекстовой разметки). Мы будем использовать его для создания наших будущих страниц. Он состоит из Тегов - это блоки которые оборачивают наш текст и придают им определенный внешний вид.
Например: <b>Hello</b>

Теги и атрибуты

Документ состоит из тегов и атрибутов.

Тег

Теги это строительные блоки нашей программы. Мы оборачиваем ими слова, давая им другой внешний вид. Например: <b>sasd</b>
