Зачем нам стили?

Мы уже видили как теги помогают нам изменять внешний вид нашей страницы. Мы с вами уже меняли размер текста, его стиль и разбивали текст на строки. Но при создании страницы, изменения внешнего вида не останавливается на изменении текста. На хороших страницах, сильно изменяются как внешний вид (фон, цвета), так и форматирование текста (отступы и расположение).

Для этого нам необхоим css: подключая его, мы получаем возможность изменять внешний вид страницы. Например.

Из чего состоит

  • link - тег обозначающий подключение к документу чего либо (в нашем слуаче стилей)
  • rel - индификатор того, что мы будем сейчас подключать у нас это stylesheet, что обозначает лист стилей.

После этих простых действий, мы подключили стили к странице, и теперь готовы изучить как именно мы можем их менять.

Синтаксис написания css

Синтаксис - это правила по которым пишутся те или иные языки. Где ставить точку, где какие скобочки открывать, и нам очен повезло. Базовый синтаксис css - очень простой.

Пример

css

        1.  p {
        2.    color: red;
        3.  }

Если обратить внимание на пример, мы увидим что применение стилей легко объясняется примером. На первой строке мы говорим что мы хотим изменить, в нашем примере это p, открываем скобки и пишем как мы хотим это изменить, в нашем примере будем менять его цвет. И важно, указываем цвет через двоеточие - :

Свойство background-color

А если нам нужно поменять фон блока, при этом любого блока, будь это вся страница body, так и просто определнного текста p

css

        1.  p {
        2.    background-color: red;
        3.  }

Свойство Border

Еще один из способов стилизовать текст, использовать границы разных цветов. Так же они бывают разных видов solid dotted. Мы должны указать три параметра: Размер в пикселях, тип границы, и цвет самой границы.

css

        1.  p {
        2.    border: 4px solid silver;
        3.  }

Свойство border-radius

После того, как мы научились украшивать рамки, есть дополнительная возможность сделать страницу лучше. Закругление углов. Использовать свойство очень просто, мы должны написать значение либо в пикселях или в процентах. Таким образом можно сделать кругю.

css

        1.  div {
        2.    border-radius: 8px;
        3.  }
        4.  div {
        5.    border-radius: 100%;
        6.  }
